## xx监控

需求：打造一个美股股票数据监控+web服务展示系统，要求轻量级。数据采集采用python，数据库采用sqlite3，web系统读取数据库数据 图表展示。



### 数据采集

- 数据范围：美股大盘（标普500、纳斯达克100、罗素2000）、及指定股票、ETF代码
- 指标范围：涨跌幅、成交额、换手率、振幅、量比、市盈率， 指标值单位转换成中文（如果支持）
- 支持补录采集指定日期范围、日常采集t+1增量数据



### web展示

- 轻量级web框架，易于部署
- 默认读取近一个月数据，支持选择日期范围
- 支持勾选多个标的、指标项以图表形式（折线、柱形图等）比较其指标
- 鼠标悬浮折线图上显示的指标值项顺序与渲染顺序一致
